## Cara Menjalankan

Setiap folder adalah project Vite terpisah. Untuk menjalankan salah satu praktik:

1. Masuk ke folder yang diinginkan:
   ```bash
   cd praktik-01
   ```
2. Install dependencies (jika belum):
   ```bash
   npm install
   ```
3. Jalankan development server:
   ```bash
   npm run dev
   ```

## Deployment

Project ini siap dideploy ke Vercel. Pastikan untuk mengatur *Root Directory* ke folder praktik yang ingin ditampilkan sebagai halaman utama, atau gunakan konfigurasi monorepo.
